OpenType ligature and feature probe
===================================

This file is meant to separate general OpenType shaping from programming-only
ligatures. A terminal that supports font shaping should preserve cell advance
while applying any enabled OpenType substitutions from the selected font.


Standard Latin ligatures
------------------------
fi   fi fi fi        office    file    final    config
fl   fl fl fl        flow      flag    inflate  reflux
ffi  ffi ffi ffi     office    affinity efficient
ffl  ffl ffl ffl     offload   waffle   suffixless
ff   ff ff ff        staff     cliff    different
ft   ft ft ft        after     often    soft
st   st st st        stack     first    last
ct   ct ct ct        action    vector   strict
Th   Th Th Th        The       That     Through


Broken Latin ligature controls
------------------------------
These should usually not become one ligature because the letters are separated.

Programming substitutions
-------------------------
These overlap with font-ligatures.txt, but are kept here to catch fonts that
implement programming forms through OpenType contextual substitutions.

arrows        ->  -->  =>  ==>  <-  <--  <=  <==  <->  <=>
compare       ==  ===  !=  !==  =/=  <=  >=  <>  <=>  ====
logic         &&  ||  !!  ?:  ??  ??=  ?.  ?:
pipes         |>  <|  ||>  <||  >=>  <=<  |->  <-|
math          ++  --  ---  **  ***  //  ///  /*  */  /**
colon forms   ::  :::  :=  ::=  =:=  =!=  =:  :==
dots          ..  ...  ....  .-  .=  .?


Context probes
--------------
The same token appears in neutral text, code-like text, and bracketed text.

neutral:       a -> b, a => b, a == b, a != b, a <= b, a >= b
code:          if (a != b && c >= d) return x => y;
bracketed:     [->] [=>] [==] [===] [!=] [!==] [<=] [>=] [<=>]
repeated:      ->->->  =>=>=>  ======  !=!=!=  <=<=<=  >=>>=
